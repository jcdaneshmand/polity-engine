# UI layout reference

Purpose: provide a Steam Deck-friendly digital board layout inspired only by functional table layout patterns.

This does **not** copy official artwork, logos, icons, fonts, or card text.

Layout map:
- Shared row (top): shared piles/decks + round summary.
- Market row (center): 5 numbered slots.
- Player area (bottom/left): status, zones, hand row.
- Right panel: selected-card detail, action menu, game log.

Steam Deck goals:
- Target 1280x800.
- Large controls and readable text.
- Keyboard/controller-friendly selection-first flow.
- No drag-and-drop dependency.

Future improvements:
- card zoom modal
- transitions/animation
- save/resume UX
- full gamepad API integration
- theme switching and accessibility presets
